# Biogas Production Analysis Using Python
**Final Year Project**  
Analyzing 60 days of biogas production data using Pandas and Matplotlib.

![Gas vs Days](gas_vs_days.png)

## Project Overview
This project performs exploratory data analysis on biogas production from an anaerobic digester.  
**No machine learning** — only data cleaning, visualization, and insights (exactly what the brief asked).

## Files in this repo
- `data/biogas_data.csv` → Clean dataset (60 days)
- `Biogas Production Analysis Using Python.ipynb` → Main Jupyter notebook
- `gas_vs_days.png`, `temp_vs_biogas.png`, etc. → All graphs

## Key Insights
- Total biogas produced: **201.49 m³** over 60 days
- Average daily production: **3.36 m³/day**
- Peak production: **Day 46 (5.17 m³)**
- Strongest factor: Substrate amount (correlation = 0.62)

## How to run
1. Clone the repo
2. Open the `.ipynb` file in Jupyter Notebook / Google Colab
3. Run all cells
