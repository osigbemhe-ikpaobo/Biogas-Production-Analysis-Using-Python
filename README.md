# Biogas Production Analysis Using Python

**Final Year Project**  
Exploratory Data Analysis of 60-day biogas production from an anaerobic digester using **Pandas** and **Matplotlib**.

![Daily Biogas Production Over 60 Days](gas_vs_days.png)

## 📊 Project Overview
This project demonstrates data cleaning, visualization, and insight generation on real-looking laboratory biogas data.  
No machine learning — only core Python skills (exactly as required for the final year project).

## 🔍 Key Insights
- **Total biogas produced**: **201.49 m³** in 60 days  
- **Average daily production**: **3.36 m³/day**  
- **Peak production**: Day 46 → **5.17 m³**  
- **Lowest production**: Day 15 → **2.08 m³**  
- **Strongest factor**: Substrate amount has the highest positive impact on biogas yield
